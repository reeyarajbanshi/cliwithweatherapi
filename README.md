I created a CLI with argparse that integrates the Weather API, has forecast, sunrise, sunset, and compare commands. It also integrates 2 APIs, one that is the Geocoding API so that the user can add in a city or zipcode and we can find out the latitude or longitude from that to input into the OpenMeteo API to get the forecast, sunrise, sunset, and compare two places. It also uses mutually exclusive groups for city or zipcode, and the compare parser has two positional arguments for two cities.
